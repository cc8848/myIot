# Node.js服务端
服务端使用基于Node.js的Express框架，[MongoDB](https://www.mongodb.com/)以及[Redis](http://redis.io/)数据库。
### 开发

##### 安装项目及依赖
```
$ git clone https://github.com/nieheyong/myIot.git
$ cd myIot/Serve
$ npm install
```
##### 运行
需要先运行[MongoDB](https://www.mongodb.com/)以及[Redis](http://redis.io/).
```
$ npm start
```
